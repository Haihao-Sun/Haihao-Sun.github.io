Leetcode-494-Target-Sum

Author: Haihao Sun Date: July 3, 2020  22:17:46 Category: DP

link to Leetcode-494-Target-Sum

Problem Description:

You are given a list of non-negative integers, a1, a2, ..., an, and a target, S. Now you have 2 symbols + and -. For each integer, you should choose one from + and - as its new symbol.

Find out how many ways to assign symbols to make sum of integers equal to target S.

Problem Discussion:

category: DP(Dynamic Programming) -> water pokemon!

Intuition:

After reading the problem description, the most straight forward solution would be using DFS and backtracking to generate all the possible combination of using + and -. Clearly, this solution is what we called NP. However, we can also use DP to solve this problem more efficiently.

Say that after processing the nums array till index i, we obtain possible solution set S_i. Namely, the next index that we are going to process is i+1. Therefore, we can obtain S_i + nums[i+1] Union S_i - nums[i+1] after processing the i + 1 element in array nums. That is how we dynamically update the solution set, and obtain the solution.
