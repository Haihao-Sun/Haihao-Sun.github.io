Leetcode-1027-Longest-Arithmetic-Sequence

Problem Description:

Given an array A of Integers, return the length of the longest arithmetic sequence in A

Example1:

Problem Discussion:

category: DP(Dynamic Programming) -> water pokemon!

Intuition:

How to find the longest subsequence that is Arithmetic in the given array A? While we always use sliding window to solve subarray problem, the common trick for solving subsequence problem is to use DP. Here we need to use store the information for the the arithemetic sequence that we can achieve in DP[i]

step1: What data structure to use to store DP information?

Essentially, we want to quickly check the longest sequence that we can achieve with difference k; clearly, we use hashmap with key : difference, and value: the length for using A[i], and the common difference for the arithemetic sequence.

Problem Solution O(n*n)
