How to watch youtube without annoying ADs?

Author: Haihao Sun Date: June 28, 2020  19:59:22

Watching Youtube without ADs

While enjoying watching Youtube, I always get annoyed by so many advertisements popping out,
and my finger has already been tired of clicking the ‘Skip’ button. Fortunately, there is a soultion to this problem,and it is super super easy.

High level description for 3-step blocking AD recipe:

step1: Downloading TamperMonkey

TamperMonkey is a script manager which helps you utilize the scripts which
are extremely handy.

step2: Downloading Hide youtube google AD script

This is a legal and safe script which allows you to skip all the ADs in
one shot. I promise you this is very easy, and no coding is involved.

step3: Turn on the script, and enjoy watching AD-free youtube!

Try your youtube without ADs!

Detailed Implementation for the recipe shown above

step1: Downloading TamperMonkey

Here is the link to the downloading TamperMonkey in chrome extension, and
this is absoultely free and safe.

Downloading TamperMonkey in google chrome

Once you open the webpage, simply add the TamperMonkey by clicking “add” at
the top right, I promise this is extremely straightforward.

step2: Downloading the Hide Youtube AD script

To download the script, we need to visit greasyforke which is a website for sharing useful scripts.

After a quick search, we can download the script here through this link
(https://greasyfork.org/en/scripts/40178-youtube-ad-skip). Click Install Script, and you should be able to see that in the TamperMonkey Extension.

You can also checkout these two similar scripts!
Auto close Youtube ADs
Youtube ADs cleaner

step3: Turn on the extension and start watching youtube without ADs!

The last step, turn on the extension, is also straightforward. Simply open google
and then find the dashboard, turn on the scipt that we just downloaded.

Congrats! ALL DONE! Enjoy your youtube without ADs!

Exciting Moment, Start watching today!
