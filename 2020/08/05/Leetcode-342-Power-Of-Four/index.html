Intuition:

We can first solve this problem by using a for loop. If a number is represented in binary, we know that a power of four should have such characteristics:

  • It can only have a single 1, with all trailing zeros.
  • The number of trailing zeros must be even. Like 16 -> 10000, 32 -> 1000000 …
